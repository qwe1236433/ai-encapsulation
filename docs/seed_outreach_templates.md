# 种子用户私信话术模板

> 用于 P3：诊断工具 MVP 的种子用户招募阶段。
>
> **基本原则**
>
> 1. **完全免费**，不设付费入口
> 2. **不承诺流量/点赞提升**（诊断工具定位是「写作级提示」，不是「爆款预测」）
> 3. 诚实披露：模型 hold-out AUC ≈ 0.53、基于健身/减脂赛道数据
> 4. **不要群发**。一次挑一个账号、阅读他最近 3 篇笔记再动手
> 5. 每封私信都要附上**这个博主最近某条笔记的具体观察**（非通用模板）

---

## 1. 招募话术（冷启动 · 给从未互动过的博主）

> **适用场景**：在小红书关键词（如「健身」「减脂」「健身餐」）里找粉丝 1k–1w 的博主。
> **发送前必做**：翻一下他最近 3–5 条笔记，挑 1 条做"观察切入点"。

**话术 A · 标准版**

```
你好～我看到你最近那条《[具体笔记标题]》写得挺用心的。

我最近做了一个小工具：基于 700+ 篇健身/减脂赛道笔记的统计分析，帮博主检查标题和正文里有没有"统计上和数据表现相关"的细节。比如：

• 标题带不带问号
• 有没有 emoji
• hashtag 数量

输出一份诊断报告，指出哪些地方可能拖了流量的后腿，附上具体的数据依据（不是玄学经验）。

免费帮你诊断 1–3 篇你之前的笔记。我只需要你粘贴标题和正文，不需要你给账号密码，也不需要装任何东西。

工具不是做"爆款预测"的，只是写作级提示——模型预测能力其实很一般（AUC 0.53），诚实告诉你。
但两条"**标题少用问号 / 加 1 个 emoji**"在我的数据里稳定性很强（Bootstrap 200 次都显著）。
想看看你的笔记有没有踩这些点吗？
```

**话术 B · 短版（适合对方最近很活跃）**

```
你好～看到你的《[笔记标题]》。我用 700+ 篇健身笔记的统计数据做了个小诊断工具，
能指出标题/正文里哪些细节和数据表现相关（免费，粘贴内容即可，无需任何授权）。

坦白说：模型整体预测能力不强（AUC 0.53），但有两条规则挺稳定的。
有兴趣让我帮你看 1 篇吗？
```

**话术 C · 回避"看起来像广告"**

```
你好，冒昧打扰。我对比了几百条健身赛道的标题结构，发现一个挺反直觉的数据：
标题带问号的笔记，高赞占比比不带问号的**低了 20 个百分点**。
我看你《[具体笔记标题]》标题是[陈述句/含问号]，恰好对上。

我做了个自动检查工具（免费）。要不要试试？纯粘贴内容，0 成本。
```

## 2. 对方回复"这工具靠谱吗"的话术

**诚实、反玄学**：

```
诚实讲：这不是一个能预测你爆款的工具，模型整体 AUC 才 0.53。
但两条规则——标题不要带问号、无 emoji 时加一个——在我 741 条样本的
Bootstrap 200 次检验里都是显著的（95% 置信区间都不跨 0）。

工具的价值是**给你一张写作检查清单**，就像程序员的 lint——
没法保证你代码跑得飞快，但能指出明显的小毛病。

如果你觉得建议不合理，大可以不听；我需要的就是你这个判断反馈。
```

**对方追问"数据哪来的"**：

```
我爬了 700+ 条公开笔记（不登录、不私信、不触犯用户隐私），
主要在健身/减脂赛道。样本偏差我也在报告末尾明确写了。
跨赛道用可能不准，你试试看结果是否对得上你的经验。
```

## 3. 对方试用后（收集反馈的话术）

**重点是收集 4 件事**，不是变现：

1. 建议是否清晰？
2. 你打算照做吗？（哪几条？哪几条不会照做？）
3. 如果你照做了，一周后告诉我流量有没有变化
4. 如果这工具收费，你觉得**最多**值多少钱？

**发送时机**：对方拿到诊断报告后 24–48 小时（别当天催）。

```
你好，想听一下你对诊断报告的反馈：

1. 建议读起来清楚吗？哪条让你看完觉得"有道理"？哪条让你觉得"纯废话"？
2. 有没有打算照着改一条试试？如果不打算改，是因为建议不合理，还是别的原因？
3. 【关键】你觉得这工具收费多少钱一次合理？或者多少钱一个月？（我只是想知道你心里价，不是真要你付钱）

哪条答得详细我都很感激——你给的数据对工具迭代很有用。
```

## 4. 如果对方主动问"能不能长期合作/要不要做产品"

**不承诺、不画饼、不签任何东西**：

```
谢谢信任。说实话现在这工具还很初级：
- 数据集还小（741 条）
- 模型准确率一般
- 还没做跨赛道验证

我想先跑 10 个免费种子用户拿数据，确认方向对了再谈付费或产品。
你愿意的话，让我帮你再诊断 2 篇，你多给点反馈，这比任何付费都更有用。
```

## 5. 常见拒绝应对

| 对方说 | 你回 |
|---|---|
| 不感兴趣/已读不回 | **不要二次骚扰**。记录下这个账号 30 天内不再联系 |
| "你是不是要卖课" | "完全没有，你看我从头到尾没提过付费/课程/引流。免费就是免费。" |
| "给个链接看看" | 发 **诊断页截图 + 页面地址**（本地跑起来后截一张示例报告） |
| "太简单了吧，这些我都懂" | "那好呀，给我反馈'哪些太简单/哪些是废话'，我就是要这种反馈来改。" |

---

## 6. 边界与红线（务必遵守）

- ❌ 不要说"爆款公式"、"一定能涨粉"、"预期提升 X%"
- ❌ 不要群发、不要在同一社群刷 5 条以上
- ❌ 不要诱导对方给账号授权/装插件/付定金
- ❌ 不要用小红书主页发"诊断工具上线啦"这种推广笔记来引流（违反私下工具属性）
- ✅ 每一次私信都要读过对方笔记、提供具体观察
- ✅ 如果对方明确拒绝，24h 内不再发消息
- ✅ 始终附上**这个工具能做什么、不能做什么**的诚实边界

## 附：观察切入点速查（健身赛道常见可诊断项）

准备私信时，先用诊断工具看他最近 3 条笔记，挑**踩雷最明显**的作为切入点：

| 观察 | 切入点话术 |
|---|---|
| 标题含问号 | "标题带问号在我的数据里高赞占比低 20pp" |
| 标题 0 emoji | "你最近几条标题都没用 emoji，数据显示加 1 个 emoji 高赞占比能高 8pp" |
| 标题 ≥ 2 个 hashtag | "标题里 3 个 hashtag，数据显示这样高赞占比反而低 17pp（可能是拖累权重）" |

这样对方**一眼能看到这工具针对他的笔记**，比通用模板回复率高多了。
